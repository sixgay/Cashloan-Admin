# 介绍

这是一个海外的现金贷金融平台，致力于帮助多家客户落地印尼、菲律宾现金贷产品

极简的 vue admin template 管理后台 提供了最基础的后台管理vue项目的搭建 ， 包含了 Element UI & axios & iconfont & permission control & lint，这些搭建后台必要的东西。 结合根据自身产品的业务需求进行开发，封装组件。

# 建议方向

  • eslint报的错误要去处理，养成良好的习惯，最简单的方式在vscode保存文件时自动修复eslint错误
  • 可使用react进行重构结合Ant Design Pro的UI库，在页面的交互和体验上与星合基本一样
  • 可使用Ant Design of Vue的UI库，在页面的交互和体验上与星合基本一样
  • 熟悉vue后台管理项目GitHub上star数较多的vue admin
  • webpack4+ 可以进一步的打包资源、开发过程优化
  • 多语言(中文|英文|印尼文)翻译的可能存在部分翻译不标准或未进行翻译
  • 路由权限vueRouter目前是后端通过json文件提供的，由于前端router根据path来进行唯一的识别，然后加载对应的component，假如path被多次赋值的话，会产生组件渲染冲突的问题，待进一步优化
  • 项目中基本上每个管理功能都会有新增和编辑时都会使用el-dialog,内嵌表单el-form的写法,由于鄙人才疏学浅，加上弹层中表单的内容随机性较多，没有想到好的封装思路，建议可以将弹层提交表单植入SearchTabelePagination中，这样的话，大部分管理的流程就可以在一个组件中实现，剩下基本就是写配置了。
