# 快速上手

注意

请确保你的 Node.js 版本 >= 8。

# Build Setup

# 1. Clone project
git clone http://.........git

# 2. Install dependencies
npm install

# 3. cp config/dev.env.js config/local.env.js
# 方便后端开发在local环境调试
npm run copyenv

# 4.0 Serve with hot reload at localhost:5000 in config/dev.env.js
npm run dev

# 4.1 Serve with hot reload at localhost:5000 in config/local.env.js
npm run local

# 5.0 Build for production with minification
# 打包
npm run build

# 5.1 Build for production and view the bundle analyzer report
npm run build --report

# 其他

### 本地生成cashloan Admin的前端技术文档
npm run docs:dev
### 打包文档
npm run docs:build
